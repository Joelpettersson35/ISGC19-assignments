# How to build, run and test assignment 2
## Cloning the repository
<ol>
  <li>Open a new terminal window</li>
  <li>Go to your prefered directory</li>
  <li>Run: <code>git clone https://github.com/Joelpettersson35/ISGC19-assignments.git</code></li>
</ol>

## Building the API
<ol>
  <li>Open Intellij IDEA</li>
  <li>On the startscreen, press "Open"</li>
  <li>Locate your cloned repository</li>
  <li>Open "assignment2" from your repository, by following this directive <code>ISGC19-assignment/backend/assignment2</code></li>
</ol>

## Running the API
<ol>
  <li>Select the file "Assignment2Application.java" under <code>src/main/java/com.group2.assignment2/</code></li>
  <li>With "Assignment2Application.java" selected, press the "play button"</li>

## Testing the API
### Postman
<ol>
  <li>To test the API we are using Postman, download at <a href="https://www.postman.com/downloads/"/>link</li>
  <li>In postman, under the collections press the "+"</li>
  <li>Choose the "REST API basics template"</li>
</ol>

### GET /books
<ol>
  <li>Click on "Get data" in your collection</li>
  <li>Unequip the query parameters if they are selected</li>
  <li>Enter this url: <code>http://localhost:8080/books</code></li>
  <li>Press send</li>
  <li>The response should contain your created books, otherwise statuscode 204</li>
  <img width="1914" height="292" alt="image" src="https://github.com/user-attachments/assets/7189c15f-2a42-484f-a9d4-2244f6d2a326" />
</ol>

### GET /books/id
<ol>
  <li>Lets try to a book by ID</li>
  <li>Paste in this url: <code>http://localhost:8080/books/id?id=YOUR_ID</code></li>
  <li>If your book exists it should be returned, otherwise a "400 bad request"</li>
</ol>

### GET /books/title
<ol>
  <li>To find by title change the url to this: <code>http://localhost:8080/books/title?title=YOUR_TITLE</code></li>
  <li>All books containing your title should be returned, otherwise a "204 no content"</li>
</ol>

### GET /books/category
<ol>
  <li>To find by category change the url to this: <code>http://localhost:8080/books/category?category=YOUR_CATEGORY</code></li>
  <li>If your book exists it should be returned, otherwise a "204 no content"</li>
</ol>

### DELETE /books/id
<ol>
  <li>Click on "Delete data" in your collection</li>
  <li>To delete your book, enter this url: <code>http://localhost:8080/books/id?id=YOUR_ID</code></li>
  <li>If the deletion was successful, "204 no content" should be returned</li>
  <li>If your book doesnt exist, this should be returned:</li>
  <img width="1910" height="316" alt="image" src="https://github.com/user-attachments/assets/b1dee812-3ab0-4f5e-9f04-b24561c48fd6" />

## Building the client
<ol>
  <li>To run the client you need node.js installed <a href="https://nodejs.org/en/download">node.js</a></li>
  <li>In your prefered IDE, open the folder "assignment2" from the frontend folder/li>
  <li>Open a new terminal window in your IDE and run: 
    <code>
      npm install
      npm run dev
    </code>
  </li>
  <li>The client application should now be up and running</li>
  <li>To access it, enter the url thats being printed in your terminal:</li>
  <img width="255" height="108" alt="Skärmavbild 2026-02-19 kl  16 20 57" src="https://github.com/user-attachments/assets/7122fac6-f214-41d8-a1d0-03cb56e8fdbf" />
  <li>Make sure to start your backend aswell</li>
</ol>
